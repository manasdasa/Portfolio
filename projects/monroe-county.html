Monroe County Fairgrounds — Website Redesign

A community UX research and consulting engagement for Monroe County's public-sector digital touchpoints — focused on accessibility, usability, and public engagement.

📅Fall 2023
🏷️Technology Consulting
💻UX Research, Figma, Journey Mapping
View on GitHub ← All Projects

The Problem

Monroe County's fairgrounds website had significant usability barriers for community members trying to book event spaces, find information, and navigate government workflows. The site was inaccessible to users with disabilities and confusing for first-time visitors.

Research Process

👥 User Interviews & Stakeholder Meetings
Pain Point Analysis & Heuristic Evaluation
User Journey Mapping
Accessibility Audit (WCAG)
Wireframing & Prototype (Figma)
✅ Redesign Recommendations Report

Key Findings & Recommendations

♿
Accessibility Gaps
Identified 12+ WCAG 2.1 violations including missing alt text, poor color contrast, and keyboard navigation failures.
🗺️
Navigation Redesign
Simplified information architecture from 6-level deep menus to a flat 3-tier structure, reducing task completion time by an estimated 40%.
📱
Mobile Responsiveness
Proposed mobile-first redesign to serve the 60%+ of users accessing the site from smartphones for event lookup.
